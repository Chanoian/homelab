# Architecture Diagrams

This folder contains editable architecture diagrams for the homelab.

Files:

- `homelab-current.mmd` - Mermaid source for the current SNO design.
- `homelab-current.md` - Markdown preview that GitHub can render as a diagram.

Edit the `.mmd` file when the infrastructure changes. The Markdown file embeds the same
diagram source for easy viewing in GitHub.
